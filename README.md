# MyPortfolio
## Tech Engineer Trainee |  Aspiring Data Scientist 

Welcome to your front row seat to my professional journey! This single-page application (SPA) is built using **React** and styled with **Tailwind CSS** to showcase my journey, projects, skills, and contact information as a Tech Engineer Trainee and aspiring Data Scientist.

-----

##  Features

This portfolio is designed to be fast, responsive, and visually appealing, offering a cohesive user experience with the following key features:

  * **Custom Cursor:** A unique, custom-designed cursor enhances interactivity and visual style.
  * **Typing Effect:** The welcome message on the **Home Page** utilizes a dynamic typing animation to greet visitors.
  * **Smooth Navigation:** Uses client-side routing (managed by React state) for instant page transitions without reloading.
  * **Responsive Design:** Optimized for seamless viewing across desktop, tablet, and mobile devices.
  * **Certificate Carousel:** The **Skills Page** includes a mechanism to cycle through multiple certificates.
  * **Functional Contact Form:** The **Contact Page** uses the `FormSubmit` service for reliable message delivery.

-----

##  User Stories

"As a Hiring Manager, I want to quickly understand the candidate's core technical skills across Cloud, AI, and DevOps so I can evaluate their fit for an entry-level or junior engineering role."

"As a Recruiter, I want the site to clearly highlight the candidate's end-to-end Capstone Project and provide working GitHub and live demo links so I can easily compile information for the technical team."

"As a Bootcamp Graduate, I want this polished, public-facing portfolio to effectively showcase my completed projects and support my job search by acting as a living resume."

##  Tech Stack

The application is built using modern front-end technologies and follows best practices for component-based architecture.

### Core Technologies

| Technology | Role |
| :--- | :--- |
| **React** | Front-end library for building the user interface. |
| **Tailwind CSS** | Utility-first CSS framework for rapid and responsive styling. |
| **JavaScript (ES6+)** | Core language for application logic and interactivity. |
| **Lucide React** | Icon library for simple and clean vector icons. |

### Architectural Structure

The application employs a well-organized folder structure to separate concerns:

```
portfolio/
├── src/
│   ├── App.jsx             ← Main state management and layout container
│   ├── components/
│   │   ├── NavBar.jsx      ← Global navigation component
│   │   └── CustomCursor.jsx← Interactive cursor component logic
│   └── pages/              ← Individual sections/views
│       ├── HomePage.jsx
│       ├── AboutPage.jsx
│       ├── ProjectsPage.jsx
│       ├── SkillsPage.jsx
│       └── ContactPage.jsx
│
└── public/
    ├── Photo.png           ← Profile picture asset
    ├── cert*.jpg           ← Certificate assets (cert1.jpg to cert11.jpg)
    └── *.svg               ← Tech stack icons (Python, SQL, AWS, etc.)
```

##  Development & Setup

To run this project locally, follow these steps:

1.  **Clone the repository:**

    ```bash
    git clone https://github.com/BM023/MyPortfolio.git
    cd portfolio
    ```

2.  **Install dependencies:**

    ```bash
    npm install
    # or
    yarn install
    ```

3.  **Start the development server:**

    ```bash
    npm run dev 
    # or 
    npm start
    ```

The application should now be running at `http://localhost:3000` (or another port depending on your setup).
